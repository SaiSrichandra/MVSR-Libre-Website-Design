About

MVSR Libre is a Free and Open Source Developers Club focuses on contributing to open-source projects to resolve local community problems and create awareness about open access among the students. MVSR Libre is collaborated with Swecha, Hyderabad. The club mission is to inspire, create and support FOSS (Free and Open Source Software).

our main motto

Motivate the students to participate in the open source projects and work on local community problems.

What is Open Source?

Open source software is a software where anyone can inspect,modify and enhance the source code.It is free for all and anyone can be a developer of free software.It can be modified to serve the needs of all kinds of users and programmers. Programmers who have access to a program's source code can improve it by adding more features to it or fixing bugs in the program.Free software witnesses regular development of the source code which is not the case with proprietary software which hides all the source code.Thus,open source software can be a great platform for students or programmers to develop better source code and use it in a variety of applications in almost all the sectors of the society.MVSR Libre - Free and Open Source Developers Club mainly focuses on creating and developing free software.

More intriguing information

MVSR Libre - Free and Open Source Developers Club is an organisation with the sole aim of creating and supporting FOSS(Free and Open Source Software). The club contributes to open source projects to resolve local community issues.It promotes the usage of free software.It creates awareness about the advantages of using open source software among the students through various projects.MVSR Libre club is now collaborated with Swecha,Hyderabad, inorder to encourage the students to know more about the implementation of free software and to make them work on projects with larger scope.The club was inaugurated on 9th April,2019 in the department of CSE in MVSR Engineering College.

So in conclusion ...

MVSR Libre - Free and Open Source Developers Club extensively promotes the creation and implementation of open source software,and helps students gain knowledge over it by conducting a variety of open source projects at different scales.The club guides the students in developing their own software from open sources and helps them use it in business applications, in solving common problems across the society etc; The organisation's main goal is to motivate students to develop and use open source software and to create awareness about it.
